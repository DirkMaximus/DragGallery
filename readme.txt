A Gallery block with Drag and Drop functionality to lay out images in rows while retaining aspect ratio.

== Description ==

A Gallery block with Drag and Drop functionality to lay out images in rows while retaining aspect ratio.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/drag-gallery` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
